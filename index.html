We run the following code to generate some local storage items. View them by clicking the plus icon ("+") in the top right hand corner.

localStorage.setItem('person', JSON.stringify({ name: 'bob', age: 20 }))
localStorage.setItem('token', '123random_token')
localStorage.setItem('valid-user', true)
localStorage.setItem('fruits', JSON.stringify(['banana', 'apple']))
